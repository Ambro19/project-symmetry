# project-symmetry
## Introduction
Semantic comparison between entire Wikipedia articles in different languages so that we can suggest edits.
## Requirements
### Python
Version 3.7.4
### NLTK
`python3 -m pip install nltk`  
### PySimpleGUI
`python3 -m pip install pysimplegui`
### Sentence Bert 
`pip install -U sentence-transformers`
